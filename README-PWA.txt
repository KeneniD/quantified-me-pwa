Quantified Me — PWA bundle (v2)

Contains:
- index.html (your app with PWA head + SW registration)
- manifest.json
- sw.js
- icons/

Deploy (GitHub Pages):
1) Create a new repo (quantified-me-pwa).
2) Upload all files in this folder (index.html, manifest.json, sw.js, icons/).
3) Settings → Pages → Deploy from branch: main / root.
4) Open the URL shown; on iPhone Safari: Share → Add to Home Screen.

Netlify: drag this folder to a new site; use the URL it gives.
